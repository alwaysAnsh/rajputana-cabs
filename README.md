# Rajputana Cabs Mhow — Website

Production-ready Next.js 14 (App Router) marketing site for **Rajputana Cabs Mhow**, a taxi service in Mhow, Madhya Pradesh.

## Scripts

- `npm run dev` — local development
- `npm run build` — production build (runs `next-sitemap` via `postbuild`)
- `npm start` — serve production build
- `npm run lint` — ESLint
